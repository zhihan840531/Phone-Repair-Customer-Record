# Phone-Repair-Customer-Record

可直接開啟 `index.html` 使用的「手機維修客戶紀錄」單頁工具。

## 功能
- 新增維修紀錄（客戶、電話、品牌、型號、維修項目、日期、狀態、售價、成本）
- 關鍵字搜尋與狀態篩選
- 自動統計筆數、總營收、總利潤
- 刪除單筆紀錄
- 使用 `localStorage` 儲存資料（留在本機瀏覽器）

## 使用方式
1. 下載或 clone 這個專案。
2. 直接用瀏覽器開啟 `index.html`。
3. 開始新增、搜尋與管理資料。

## 注意
- 目前是前端本機版，沒有後端資料庫。
- 若要「直接進 GitHub 更改」，可在 GitHub 網頁編輯檔案後 commit；
  或用本地端改好後 push 到你的 GitHub repo。
